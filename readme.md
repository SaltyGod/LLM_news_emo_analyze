# 新闻情感分析系统

## 项目简介
这是一个基于大语言模型的新闻情感分析系统，主要用于分析新闻文本的情感倾向和主题分类。该系统集成了BGE中文嵌入模型和情感词典，可以对新闻文本进行多维度的分析。

## 主要功能

### 1. 新闻时效性分析
- 分析新闻发布时间与事件发生时间的滞后关系
- 支持多种时间表达方式的识别（具体日期、"昨天"、"前天"、"近日"等）
- 统计分析新闻发布时间分布特征

### 2. 新闻文本分析
- 文本长度统计分析
- 新闻内容提取和处理
- 最大文本长度限制：2000字符

### 3. 情感分析
- 基于情感词典的情感分析
- 支持英文情感词典（Bignomic词典）
- 情感词典包含词语及其对应的情感分数

### 4. 主题匹配
- 使用BGE-large-zh模型进行文本嵌入
- 计算文本相似度进行主题匹配
- 支持自定义主题匹配规则
